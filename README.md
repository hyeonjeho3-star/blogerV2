# Blog Mate v2.0 - Smart Keyword Discovery Platform

블로그 성공을 위한 스마트 키워드 발굴 플랫폼

## 주요 기능

- 🔍 **스마트 키워드 자동 발굴**: 관심 분야만 입력하면 S급 키워드 자동 추천
- ⚖️ **키워드 트렌드 비교**: 네이버 데이터랩 기반 키워드 비교 분석
- 🎯 **전략 리포트**: 데이터 기반 블로그 발행 전략 제시

## 기술 스택

- Frontend: Streamlit 1.28+
- Backend: Python 3.10+
- APIs: Naver DataLab, Google Gemini
- Testing: pytest
